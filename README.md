# WASD Range

Простая 3D стрелялка в браузере: большая арена, плавное WASD-движение, спринт, raycast-выстрелы, здоровье, рекорд по убитым зомби и волны ЕГЭ-зомби.

## Запуск без Node.js

Дважды кликни:

```text
start-game.cmd
```

Или запусти из PowerShell:

```powershell
.\start-game.cmd
```

Скрипт сам откроет браузер на `http://127.0.0.1:4173/`.

## Управление

- WASD - движение
- Мышь - обзор
- ЛКМ - выстрел
- Shift - бег
- V или кнопка в правом верхнем углу - переключить вид
- P или кнопка `⏸` - пауза
- 1-9 и 0 - выбрать или купить оружие
- Esc - пауза через отпускание курсора

На телефоне:

- левый стик - движение
- свайп по экрану - обзор
- большая кнопка справа - атака
- кнопка `⇧` - бег

## Цель

Переживай волны зомби. Они бегут к игроку и наносят урон рядом. За убийства и зачистку волн дают деньги, на них можно покупать лук, меч, дробовик, автомат, арбалет, снайперку, пулемет, рельсу и миниган. После зачистки волны здоровье немного восстанавливается.

В стартовом меню можно выбрать, против какого ЕГЭ сражаться: математика, физика, русский язык, обществознание, химия, биология или английский. Математика использует `download.png`, остальные обложки лежат в `assets/`.

Саундтрек: `godlikelove - Die Wut`, файл `assets/soundtrack.mp3`. Музыка запускается после нажатия `Начать` или первого тапа в игре.

## Запуск через Node.js

```powershell
node server.mjs
```

После запуска открой:

```text
http://127.0.0.1:4173/
```

Игра использует Three.js с CDN, поэтому браузеру нужен доступ к `unpkg.com`.
